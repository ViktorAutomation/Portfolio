FB & LinkedIn Auto-Posting via Custom GPT Action

An AI-powered content pipeline that turns news articles into ready-to-publish posts for Facebook and LinkedIn — all controlled from a single Custom GPT conversation and automated end-to-end with Make.com.

Custom GPT Actions Make.com Facebook API LinkedIn API DALL·E 3 Airtable

🎯 The Challenge

The client was tired of manually searching for articles, copy-pasting links between tools, and rewriting content for each platform. They wanted a consistent, repeatable posting flow across Facebook and LinkedIn — and a way to track what got published, when, and where, without jumping between five different tabs to do it.

Goal: a single-conversation workflow that could search, curate, create, publish, and track social posts with minimal manual effort.

🧩 The Solution

Custom GPT Actions become the control center where the user chats naturally; Make.com scenarios handle the heavy lifting in the background. The user never leaves the GPT conversation.

01
Custom GPT Actions as the control center
GPT acts as the front-end where the user chats naturally. Actions call Make.com scenarios with the user's choices — topics, platforms, edits.
02
News discovery & content curation
Make.com scenarios parse predefined news sources and websites. Curated article options (title, link, summary) are passed back into the GPT conversation for the user to choose from.
03
Content selection & editing inside GPT
The user selects an article and refines the post copy directly in the conversation. GPT generates platform-ready text tailored to LinkedIn and/or Facebook tone.
04
Image handling — original or AI-generated
If available, the article's original image is reused. Alternatively, GPT triggers OpenAI DALL·E 3 via Make.com to generate a new, on-brand image.
05
Automated posting to social platforms
Final content (text + image) is pushed through Make.com to:
  • Facebook Pages via API
  • LinkedIn pages/profiles via API
06
Tracking & logging in Airtable
Every successful post is recorded in Airtable (article URL, platforms, timestamps, copy used), creating an analytics-ready dataset for review and reporting.

📈 Results & Impact

  • End-to-end workflow in a single GPT chat — content sourcing, creation, posting, and tracking all run from one conversation.
  • Time saved — manual multi-tool workflow reduced to a few clicks and short prompts.
  • Better consistency — regular posting schedule with a consistent structure and brand voice across both platforms.
  • Improved content quality — AI-assisted copy and images raise the baseline of every post.
  • Clean analytics dataset — Airtable log of every post enables quick review of what shipped, where, and when.
